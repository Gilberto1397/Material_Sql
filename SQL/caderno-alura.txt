Definições
    A entidade principal é o próprio DB armazenando tabelas

    Ao se criar a tabela definimos o tipo dos campos e o nº de campos
    As linhas podem ser infinitas só o nº de colunas que é estipulado
        Podemos dizer quanto de disco esse banco poderá ter

    Chave primária: Pode ter não é obrigatório e não permiti que este campo se repita
        Chave primária composta - para 2 ou mais colunas

    Chave estrangeira: Interliga diferentes tabelas fazendo com que o dado preenchido exista na tabela que envia a chave
        ---Gera índice automaticamente

    índice: instrumento usado para achar elementos na minha tabela de maneira mais rápida        
        Exemplo: o índice meio que gera uma pré organização da coluna com índice, para quando rodar uma busca por um nome
        ele ajuda levando em consideração a ordem alfab. assim a busca se inicia de acordo com a letra do nome buscado

    Esquemas: são grupos de tabelas cujos dados envolvem o mesmo assunto

    View: Fundiona como uma tabela, porém apenas é uma query sendo executada automaticamente entregando os dados
        Pode conter uma regra de negócio
